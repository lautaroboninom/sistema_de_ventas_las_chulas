Branding Equilux:

- logo-empresa.png -> Logo corporativo oficial (UI/PDF).
- logo-app.png -> Icono base de la app (E blanca sobre fondo celeste).

Generacion de iconos (desde `web/`):

```
npm run icons
```

El comando genera en `public/icons/`:
- logo-app-16.png, logo-app-32.png (favicons)
- logo-app-180.png (apple touch)
- logo-app-192.png, logo-app-512.png (PWA)
- logo-app-512-maskable.png (PWA maskable)
